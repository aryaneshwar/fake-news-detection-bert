# Fake News Detection using BERT

## 📌 Problem Statement
Fake news detection is a critical NLP task. The goal is to classify news as Fake or Real using transformer models.

---

## ⚙️ Approach
- Data preprocessing and analysis
- Tokenization using BERT tokenizer
- Fine-tuning pretrained BERT model
- Evaluation using classification metrics
- Error analysis to understand model limitations
- Model improvement using DistilBERT comparison
- Deployment using Gradio

---

## 🤖 Models Used
- bert-base-uncased
- distilbert-base-uncased

---

## 📊 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix

---

## 🚀 Results
- Achieved high accuracy using BERT
- DistilBERT provided faster inference with slight trade-off

---

## 🔍 Key Learnings
- Understanding transformer architecture
- Importance of tokenization
- Fine-tuning pretrained models
- Error analysis in NLP
- Model deployment
